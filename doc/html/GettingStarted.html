Getting Started with regularizeNd

Description

regularizeNd toolbox is a set of tools to create a gridded lookup table of scattered data in n dimensions.

Features

  • regularizeNd function calculates a gridded lookup table from scattered data in n dimensions.
  • regularizeNdMatrices function returns the matrices used to in calculating a gridded lookup table from scattered data. This is useful for solving with constraints.
  • lsqConstrainedAlternative is alternative lsqlin for solving linear least square problems with inequality constraints.

Examples

See examples folder.

Documentation

See doc folder or built-in documentation.
To use the built-in documentation, open the main help page (F1). Look for "regularizeNd Toolbox" at the bottom of the page under the Supplemental Software section. If regularizNd is not in that section, you can regenerate the documentation database by using builddocsearchdb(). Do the following:
  1. Ensure that this toolbox is on the path and contains info.xml. Make sure the current path is not the "doc" folder.
  2. Call with the full path to the "doc" folder where the help files are contained and helptoc.xml. builddocsearchdb('C:\pathToToolbox\doc'); Note, it has to be the full absolute path. Using a relative path will not work.
  3. Check the Supplemental Software section of the main help page (F1).
